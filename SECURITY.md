# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Vaelor, please report it
**privately** by opening a [GitHub Security Advisory](https://github.com/anatolykoptev/vaelor/security/advisories/new).

Do **not** open a public issue for security problems.

We aim to acknowledge reports within 72 hours and provide an initial
assessment within 7 days. Coordinated disclosure is preferred — we will
work with you on a timeline before any public announcement.

## Supported Versions

Only the latest minor release receives security fixes.
